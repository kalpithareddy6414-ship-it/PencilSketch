# Pencil Sketch Generator (Python + OpenCV)
A simple Python project that converts a normal image into a pencil sketch using OpenCV.  It mimics hand-drawn sketches by applying grayscale conversion, inversion, blurring, and division techniques.

**Features**
- Convert any image into a pencil sketch
- Fast and lightweight
- Uses OpenCV for image processing

  **Tech used**
- Python
- OpenCV (cv2)

 **Installation**
 pip install opencv-python

 **How to run**
 python sketch.py
